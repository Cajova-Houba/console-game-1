# About

Simplistic bomberman-like game. Demolish walls and pickup diamnods.

# Map description

 - ' ' (space): Empty space
 - 'W': Wall
 - 'D': Diamond
 - 'P': Player
 - 'B': Bomb
 - 'X': Explosion (not dangerous for player)

# Controls

 - Movement: WSAD
 - Plant bomb: Space
 - Next level: N
 - Quit: ESC
